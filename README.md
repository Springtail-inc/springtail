# Springtail

## Pre-requisites:
- Install Postgres 16
  - Mac: `brew install postgresql@16`
  - Make sure pg_config is in the path, I added: `/opt/homebrew/opt/postgresql@16/bin`
- Conan -- not currently required(C++ package manager)
  - install through pip3 `pip3 install conan`
  - make sure conan binary is in your path
